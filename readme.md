### Welcome to the Magic Login for Paid Memberships Pro GitHub Repository
Adds passwordless login via secure email links to Paid Memberships Pro.

For more information please visit [paidmembershipspro.com/add-ons/magic-login](https://www.paidmembershipspro.com/add-ons/magic-login)

## Installation ##
For detailed installation steps, visit the [documentation](https://www.paidmembershipspro.com/add-ons/magic-login) page.

1. Download the current development ZIP file directly: `https://github.com/strangerstudios/pmpro-magic-login/archive/dev.zip`

**Please ensure that once installing this version of the plugin to remove `-dev` from the plugin's folder name.**

## Bugs ##
If you find an issue/bug, let us know by [creating a detailed GitHub issue](https://github.com/strangerstudios/pmpro-magic-login/issues/new).

## Support ##
This is a developer's portal for Magic Login for Paid Memberships Pro. We do not offer support on this channel. **Any support related questions should be directed to [paidmembershipspro.com/support](https://www.paidmembershipspro.com/support/).**

## Contributing to Magic Login for Paid Memberships Pro ##
We encourage and welcome any contribution to Magic Login for Paid Memberships Pro. Please read the [guidelines for contributing](https://github.com/strangerstudios/pmpro-magic-login/blob/dev/.github/CONTRIBUTING.md) to this repository.

There are various **ways to the help development** of Magic Login for Paid Memberships Pro:

1. Report [bugs/issues](https://github.com/strangerstudios/pmpro-magic-login/issues/new) on GitHub.
2. Work on any issues by submitting a Pull Request.

Here are some ways for **non-developers to contribute** to Magic Login for Paid Memberships Pro:

1. Translate Magic Login for Paid Memberships Pro into your own [language](https://www.paidmembershipspro.com/paid-memberships-pro-in-your-language/).
2. [Purchase a paid membership](https://paidmembershipspro.com/pricing) to help fund ongoing development and bug fixes.
3. Leave an honest review for [Magic Login for Paid Memberships Pro](https://www.paidmembershipspro.com/submit-testimonial/).
